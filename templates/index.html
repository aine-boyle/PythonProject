Twitter Sentiment Analysis




About This Project

I have implemented a system which performs Sentiment Analysis of Twitter data.

A user enters a keyword in the user interface. This keyword is used as a parameter to scrape Twitter and find relevant Tweets. These Tweets are then classified and displayed to the user in a way that is easy to understand.

The user is provided with both a simple, clear overview, as well as a more detailed breakdown of the results
